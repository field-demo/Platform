# AMOS Field Interface - PWA Prototype

Prototype for a mobile field interface that supports preventive maintenance inspection rounds.

## Features

- Mobile-first PWA layout
- Local tag search
- AMOS ID search
- Frequency filters: 1W / 1M / 4M
- Free inspection workflow
- Status updates without signature
- Comments and severity
- Findings overview
- Management dashboard
- Simulated sync
- Offline-ready demo using local storage

## Concept

Ex-approved phone → Internal PWA → Controlled backend/middleware → AMOS integration

AMOS remains the official maintenance system. This prototype is a field execution interface intended to make inspection rounds easier, faster and more traceable.
